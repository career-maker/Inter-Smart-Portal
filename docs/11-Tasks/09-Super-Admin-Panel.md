# Super Admin Panel

Project

Intersmart Employee Portal

Version 1.0

---

# Purpose

The Super Admin Panel provides complete administrative control over the Employee Portal.

The Super Admin shall manage:

- Employees
- Teams
- Team Leads
- Leave Policies
- Leave Balances
- Holidays
- Document Requests
- HR Policies
- Announcements
- View The Hall
- Reports
- System Settings
- Roles & Permissions
- Dashboard Widgets
- Audit Logs

The Super Admin has unrestricted access to all modules.
