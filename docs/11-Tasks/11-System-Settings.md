# System Settings

Project

Intersmart Employee Portal

Version 1.0

---

# Purpose

The System Settings module provides centralized configuration for the Employee Portal.

Only Super Admin shall have access to this module.

All configurable values throughout the application shall be managed from this section.

Changes made here shall take effect immediately unless otherwise specified.
