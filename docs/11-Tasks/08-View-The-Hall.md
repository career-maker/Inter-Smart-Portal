# View The Hall

Project

Intersmart Employee Portal

Version 1.0

---

# Purpose

View The Hall provides a real-time company-wide availability dashboard.

The module enables Team Leads and Super Admin to quickly view employee availability across all teams.

Employees shall not have access to View The Hall.

Only the following roles may access this module:

- Team Lead
- HR
- Super Admin

The dashboard shall display:

- Employees Working
- Employees on Leave
- Employees Working From Home
- Half-Day Leave
- Half-Day WFH
- Pending Leave
- Disabled Employees

All information shall be updated automatically whenever leave requests or WFH requests are approved or rejected.
