# Document Request Management

Project

Intersmart Employee Portal

Version 1.0

---

# Purpose

This module enables employees to request official HR documents.

Employees can:

- Select a document
- Submit request
- Track status
- Receive notification
- Download completed document

HR receives the request by:

- Dashboard
- Email
- Notification Center

HR uploads the completed document.

Employee receives:

- Email Notification
- Website Notification

Completed document becomes available under Downloads.

---

# Roles

Employee

Can

- Request Documents
- View Status
- Download Completed Documents

Same permissions as Employee.

- View Requests
- Upload Documents

- Add Notes

- Requests
