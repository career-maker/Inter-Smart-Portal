# Updates & Announcements

Project

Intersmart Employee Portal

Version 1.0

---

# Purpose

This module manages company-wide updates, announcements, celebrations, and employee engagement.

The system shall display announcements as popup banners, dashboard widgets, and notification cards.

Employees shall be able to interact with birthdays and work anniversaries by sending wishes.

---

# Roles

Employee

Can

- View announcements
- View birthdays
- View work anniversaries
- Send wishes
- View HR policies

---

Team Lead

Same permissions as Employee.

---

HR

Can

- Create announcements
- Publish birthday posters
- Publish work anniversary posters
- Manage HR policies

---

Super Admin

Full access to manage:

- Announcements
- Categories
- Posters
- Scheduling
- Employee wishes
- Visibility
- HR Policies
